# Invan

Tovar ishlab chiqaruvchilar uchun ichki tizim. Tizim tovarlarni nazorat qilish uchun tayyorlangan.

## Technologies [Backend]

**Library**: Express.js \
**Database**: MongoDB \
**ORM**: Mongoose

## Installation

Dependencies:
- Node.js
- Npm
- MongoDB

**Start server**

```bash
npm install
node server
```


**Backend**: [localhost:5000](http://localhost:5000)
